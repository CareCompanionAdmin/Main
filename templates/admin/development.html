Quick Start Guide

1 Enable Dev Mode

Enter your public IP address and click Enable. This opens SSH port 22 in the AWS security group for your IP only.

2 Download SSH Key

Download the .pem file (Linux/Mac) or .ppk file (Windows/PuTTY). Set permissions: chmod 600 key.pem

3 Connect via SSH

Use the SSH command shown above. When done, disable dev mode to revoke access and terminate sessions.

Security Notice

Development mode opens SSH access to production servers. Always disable when not actively using. Only one IP address can be whitelisted at a time. Disabling will terminate all active SSH sessions.
